Виды селекторов

Селекторы делятся на такие типы:

  • Селектор по названию элемента h1. Этот селектор выбирае все элементы с тэгом h1
  • Селектор class элемента .classname. Этот селектор выбирает элементы за нужным ему классом
  • Селектор id элемента #headding. Этот селектор выбирает элементы за нужным ему классом, но он может исспользовать единожды этот элемент и только 1 id
  • Cелектор группирования элементов h1,h2,h3,p. Всем этим элементам, которые перечисляются через запятую, могут использоваться одни и теже свойства элемента/li>
  • Мультиклассы .class1.class2. К любому элементу одновременно можно добавить несколько классов, перечисляя их в атрибуте class через пробел
  • Вложенные селекторы p h2 /h2 /p.В селекторе работают свойства для внутреннего элемента h2
  • Универсальный селектор *. Селектор распространяется на все элементы
  • Селектор по атрибуту blockquote[cite]. Селектор по атрибуту находит элемент на странице по значению либо по наличию атрибута
  • Селектор дочерних элементов div > span. Комбинатор > разделяет 2 селектора, находит элементы заданные вторым селектором, являющие прямыми потомками для элементов отобранных первым селектором
  • Сестринские селекторы i+b. Стиль при такой записи применяется к элементу b, но только в том случае, если он является соседним для элемента i и следует сразу после него.
  • Родственные селекторы h1~p. Тоже самое, что и в сестринском селекторе только применяется ко всем элементам выбранного типа
  • Селекторы атрибутов [<атрибут>]. Устанавливает стиль для элемента, если задан специфичный атрибут
  • [attr^='value'] [атрибут^="<значение>"]. Устанавливает стиль для элемента в том случае, если значение атрибута начинается с указанного текста
